# Project Brief

## Core Description
Black Book 是一个 Local-First、AI 驱动的极简人脉情报系统，专为高净值人群设计。应用采用零知隐私架构，所有数据仅存储在本地加密 SQLite 数据库中，通过生物识别（FaceID/TouchID）保护，确保"关系比钱更重要"的核心用户群体能够安全地管理和追踪他们的核心人脉网络。

## Key Features
- **Zero-Knowledge Privacy (零知隐私)**：数据只存在于设备加密芯片中，无云端存储
- **Voice-First Input (语音优先录入)**：超大录音按钮，支持离线音频录制和语音转文字
- **AI Intelligence Analysis (AI 情报分析)**：规则引擎自动提取人名、健康、需求、偏好等实体
- **CIA-Style Dossiers (CIA 风格档案)**：精美的人物卡片，追踪健康、偏好、禁忌和交往时间轴
- **Kill Switch (一键自毁)**：紧急情况立即销毁所有数据
- **Biometric Security (生物识别锁)**：FaceID/TouchID 应用锁定

## Target Audience
- 需要管理 150 个核心利益相关者（而非 5000 个泛泛之交）的精英阶层
- 重视隐私和数据安全的高净值个人
- 需要通过语音快速记录商业情报的商务人士

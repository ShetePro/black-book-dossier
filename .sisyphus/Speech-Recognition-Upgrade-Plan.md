## 技术实现细节

### 1. 讯飞 WebSocket API 接入

**连接地址**: `wss://iat-api.xfyun.cn/v2/iat`

**鉴权方式**:
```
Date: RFC1123 格式日期
Authorization: Base64(HMAC-SHA256(api_secret, signature))
```

**请求参数**:
```json
{
  "common": {
    "app_id": "YOUR_APP_ID"
  },
  "business": {
    "language": "zh_cn",
    "domain": "iat",
    "accent": "mandarin",
    "dwa": "wpgs",        // 动态修正
    "pd": "business"      // 商务领域
  },
  "data": {
    "status": 0,          // 0=开始, 1=继续, 2=结束
    "format": "audio/L16;rate=16000",
    "encoding": "raw",
    "audio": "base64..."
  }
}
```

### 2. 流式识别优化

**分片策略**:
- 每 200ms 发送一次音频数据
- 实时返回中间结果
- 最终返回修正后的完整结果

**动态修正**:
- 开启 `dwa: "wpgs"` 参数
- 服务端会多次返回结果，逐步修正
- 最终结果的准确率最高

## 风险与对策

| 风险 | 对策 |
|------|------|
| 网络不稳定导致识别失败 | 实现自动降级到本地 Whisper |

| 讯飞服务不可用 | 接入阿里云作为备选 |

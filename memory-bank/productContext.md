# Product Context

## Why this project exists
现代社交网络工具要么过于公开（LinkedIn、微信），要么过于复杂（CRM 系统），没有一款工具专注于：
1. **极致隐私** - 让高净值人群放心存储敏感的人脉信息
2. **语音优先** - 在商务场合快速记录关键情报，无需打字
3. **精选网络** - 管理真正重要的核心关系，而非海量联系人
4. **智能分析** - 自动从对话中提取有价值的商业情报

Black Book 填补了这个空白，为精英阶层提供一个"数字黑皮书"，让他们能够像情报机构一样管理人脉关系。

## User Experience Goals
- **极致隐私体验**：用户完全掌控自己的数据，无需担心泄露
- **语音优先界面**：单手操作，3 秒内开始录音
- **环境智能**：酒后胡言乱语也能转化为结构化情报
- **精美视觉**：Dark Mode Only，金色强调色，营造奢华感
- **触感反馈**：Haptic Feedback 增强操作的确定性
- **零学习成本**：直观的 CIA 风格档案卡片，一目了然
